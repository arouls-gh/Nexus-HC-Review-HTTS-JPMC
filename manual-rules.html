Template keywords

Use keywords in static feedback to pull variables from HC report's message block in runtime

{{message}} First line from the message column.

{{first_message}} Same as the first message line.

{{message_lines}} All message-column lines shown line by line.

{{messages}} Same as all message-column lines.

{{message_pipe}} All message-column lines joined with |.

{{check_name}} Current health-check name.

{{hostname}} Switch hostname from the uploaded folder/report.

{{interfaces}} Interfaces found in the message block.

{{compact_interfaces}} Interfaces in compact range format like Ethernet1/5-6.

{{vrfs}} VRF names found in the message block.

{{neighbor_ips}} BGP neighbor IP addresses found in the message block.

{{ip_addresses}} IP addresses found in the message block.

{{description_lines}} Lines from the description column for that check block.

{{command_hint}} Command hint shown in the report for that check block.

{{module_ids}} Module identifiers found in the message block.

{{utilization}} Utilization value or percentage found in the message block.

{{class_maps}} CoPP or class-map names found in the message block.

{{bug_ids}} Bug IDs found in the message block.

{{classification}} Classification of the current manual rule.
